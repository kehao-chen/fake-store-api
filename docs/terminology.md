## 使用指南

1. **優先使用建議術語**：在撰寫文件時，應優先使用「建議使用」欄位中的術語。

2. **保持一致性**：在同一份文件或專案中，術語使用應保持一致。

3. **技術名詞保留英文**：框架、工具、服務等專有名詞保持英文原文。

4. **上下文考量**：某些術語可能根據上下文有不同的翻譯，應選擇最適合的版本。

5. **定期更新**：隨著專案發展，持續更新和完善術語表。
